A python client for Openfire's REST API Plugin

Installation
----------------
Install from PyPI:

        pip install openfire-restapi

Install from source:

        $ git clone git://github.com/allianceauth/openfire-restapi.git
        $ cd openfire-restapi
        $ python setup.py install
